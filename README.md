# 🚀 NavBar Responsiva e Acessível

Projeto desenvolvido com foco em resolver um problema real do mercado: criar uma navegação responsiva, moderna e acessível para dispositivos mobile e desktop.

## 📌 Sobre o Projeto

Esse projeto consiste em uma **Navbar Responsiva com Menu Hambúrguer**, Tecnologias Utilizadas


- HTML5
- CSS3
- JavaScript 

A aplicação adapta automaticamente o menu para diferentes tamanhos de tela, oferecendo uma experiência fluida tanto no desktop quanto no mobile.

---

## 🎯 O Problema

Muitos sites possuem menus que:
- quebram em telas menores;
- oferecem uma experiência ruim no mobile;
- não possuem acessibilidade adequada;
- dificultam a navegação do usuário.

Como desenvolvedor Front-End em evolução, decidi criar uma solução prática para estudar:
- responsividade;
- estruturação semântica;
- manipulação de DOM;
- acessibilidade;
- experiência do usuário (UX).

---

## 💡 A Solução

Desenvolvi uma Navbar Responsiva que:

✅ Exibe links horizontais no desktop  
✅ Mostra menu hambúrguer no mobile  
✅ Possui animação suave na abertura  
✅ Utiliza Flexbox para alinhamento moderno  
✅ Funciona com JavaScript puro  
✅ Tem estrutura preparada para acessibilidade  

---

## 📚 O Que Aprendi

Durante o desenvolvimento deste projeto, pratiquei:

- Estrutura semântica com HTML
- Responsividade com Media Queries
- Flexbox
- Manipulação de classes com JavaScript
- Eventos com `addEventListener`
- Organização de código Front-End
- Construção de interfaces mobile-first

---

## ⚙️ Funcionamento

### Desktop
- Links aparecem horizontalmente.

### Mobile
- O menu é ocultado.
- O botão hambúrguer aparece.
- Ao clicar, o menu desliza suavemente para a tela.

---
